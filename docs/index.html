What it does

Everything you need, nothing you don't

Built for writers who want their work safe in GitHub — without ever touching a terminal.

✍️

WYSIWYG Markdown Editor

A single continuous document surface — Word/Docs-style. Headings, bold, italic, code, underline, and strikethrough rendered in real time. No raw syntax visible while editing.

☁️

Push on Every Save

Press Ctrl+S and your document is pushed to GitHub automatically. No staging, committing, or pushing. The status badge in the corner tells you it worked.

📂

File Explorer Sidebar

Browse and manage all your Markdown files in a tree view. Create files and folders. Rename them. Everything reflects on GitHub immediately.

🔷

Mermaid Diagrams

Write Mermaid code blocks and see full-color flowcharts and sequence diagrams rendered inline — no browser, no Node.js, pure Rust renderer.

🔗

Multi-Repo Support

Connect multiple GitHub repositories. Each local folder maps to a different repo, branch, and folder — all managed from one app.

⚡

60-Second Onboarding

On first launch, the app walks you through GitHub login via OAuth device flow and repo selection. You're writing in under a minute — no configuration screens.

How it works

As simple as four steps

1

Open OctoDocs

The app walks you through connecting your GitHub account via OAuth device flow — no passwords, no SSH keys.

2

Pick a repo & folder

Choose where your documents live. OctoDocs imports any existing .md files from that folder automatically.

3

Write

Use the WYSIWYG editor. Formatting is rendered as you type. No Markdown syntax visible unless you want it.

4

Save

Ctrl+S saves locally and pushes to GitHub in the background. A status badge confirms the sync.

Get started

Download OctoDocs

Free and open source. Pre-built binaries for Linux, macOS, and Windows (experimental).

🐧
Linux
x86_64
curl -f https://raw.githubusercontent.com/elranu/octodocs/main/install.sh | sh

Installs to ~/.local/octodocs/ and adds octodocs to your PATH.

↓  Download .tar.gz
🍎
macOS
Apple Silicon (aarch64)

Download the DMG, open it, and drag OctoDocs to /Applications.

Unsigned app. On first launch, right-click → Open, or go to System Settings → Privacy & Security → Open Anyway.
↓  Download .dmg
🪟
Windows
x86_64 · Experimental

Download the ZIP, extract it, and run octodocs-app.exe.

Unsigned binary. SmartScreen will warn you. Click "More info" → "Run anyway" to launch.
↓  Download .zip

Build from Source

Requires Rust (nightly — selected automatically via rust-toolchain.toml).

git clone https://github.com/elranu/octodocs.git cd octodocs/desktop cargo run -p octodocs-app

Linux also needs: libxcb1-dev libxkbcommon-dev libwayland-dev libvulkan-dev libsecret-1-dev libfontconfig-dev libasound2-dev libssl-dev
